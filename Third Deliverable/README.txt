Guida all’installazione del PPC su server virtuali WAMP, MAMPP o XAMPP

1) Copiare la cartella “Prototipo” all’interno della cartella “www” (Windows) o “htdocs” (Mac)
2) Creare un database di nome “ingegneriasoftware” tramite phpMyAdmin
3) Importare il file “SQL.sql” presente nella cartella “Prototipo”
4) Avviare il PPC scrivendo “localhost/Prototipo” sull’url
